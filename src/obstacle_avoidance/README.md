# Obstacle Avoidance Package

This package provides nodes and utilities for obstacle avoidance in the robot system.

## Structure
- `cfg/`: Configuration files for obstacle avoidance parameters.
- `include/obstacle_avoidance/`: C++ header files.
- `launch/`: Launch files for starting nodes.
- `rviz/`: RViz configuration files.
- `scripts/`: Python scripts for utilities or nodes.
- `src/`: C++ source files.
